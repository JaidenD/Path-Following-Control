## Usage
Examples of usage

```bash
SIM_MODE=approximate SHOW_PLOT=1 T_FINAL=5.0 python3 Examples/two_link_manipulator.py
```

Run approximate simulation:

```bash
SIM_MODE=approximate python3 Examples/two_link_manipulator.py
```

Run exact simulation:

```bash
SIM_MODE=exact python3 Examples/two_link_manipulator.py
```

To show plots:

```bash
SHOW_PLOT=1 python3 Examples/two_link_manipulator.py
SIM_MODE=approximate SHOW_PLOT=1 python3 Examples/two_link_manipulator.py
```
